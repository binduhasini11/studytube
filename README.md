StudyTube AI 🎓

An AI-powered YouTube video summarizer built for students, researchers, and curious learners.

StudyTube AI transforms long YouTube videos into smart summaries, key takeaways, and structured learning notes tailored to your purpose — whether you're learning a new topic, doing research, studying educational content, or exploring stories.

🌟 Vision

Turn passive video watching into active learning.
StudyTube AI helps students save time, revise faster, and focus on understanding instead of endlessly rewatching long videos.

🚀 Features

- 🎥 Paste any YouTube video link
- 🧠 AI-generated smart summaries
- 📌 Key takeaways & revision points
- 📚 Related reading suggestions
- 🎯 Purpose-based summaries:
  - Learn
  - Research
  - Educational
  - Story
  - Other
- 🌙 Minimal dark-mode student-focused UI
- ✨ Interactive and modern design
- 📱 Fully responsive
